Principais Formatações

Negrito / Destaque

Algum termo em negrito utilizando a tag <b> (não semantica)

Algum termo em destaque utilizando <strong> (semantica)

Itálico / Ênfase

Alguem termo em Itálico usando a tag <i>, mas tambem posso selecionar o termo em destaque e utilizar Ctrl+P para utilizar o Wrap with abbreviation e selecionar a tag (não semantica)

Algum termo em Ênfase utilizando a tag <em>, mas tambem posso selecionar o termo em destaque e utilizar Ctrl+P para utilizar o Wrap with abbreviation e selecionar a tag (semantica)

Texto Marcado

Podemos criar tambem um texto marcado usando a tag MARK

Texto deletado

Texto deletado é basicamente um texto que pode ser lido mas no final ele não importa

Texto Inserido

Texto Inserido é basicamente um texto que serve para dar enfase e indicar que ele foi adicionado depois (não semantica)
